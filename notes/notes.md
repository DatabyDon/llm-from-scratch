# 1. The Setup

Setting up the environment was pretty straight forward. This is the first time I ran into the uv package and choosing the python version. Ultimately I chose python 3.12 because... Claude told me to. Here's what it said:  

```
For a learning project like this, I'd go with Python 3.12 rather than jumping to the latest.  
  
Here's the reasoning:
The current stable releases are 3.13 and 3.14, but for ML/AI work the sweet spot right now is 3.12 because PyTorch, NumPy, and the rest of the ML ecosystem have had time to fully stabilize on it. Bleeding edge versions like 3.14 sometimes have library compatibility lag — packages haven't caught up yet, and you'll hit weird install errors that have nothing to do with what you're actually trying to learn.
```  
So that's what I did.
